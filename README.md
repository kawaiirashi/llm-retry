## 📘 What is llm-retry?

llm-retry is a small software tool that helps programs work better when they talk to large language model (LLM) services. Sometimes these services get busy, slow down, or stop working for short times. llm-retry helps manage those problems by trying again, waiting when needed, and switching to backup options automatically.

This makes apps that use LLMs less likely to crash or stop working due to temporary errors like message limits or delays.

You do not need to know programming to benefit from this tool if your software supports it. It is built to work quietly in the background.

## 🔧 Configuration (Optional)

While llm-retry works without extra setup, some users can adjust settings such as:

- Retry limit: How many times it tries again after failure
- Timeout length: How long to wait before stopping a request
- Failover order: The order to try different API providers if one fails

These settings usually require modifying configuration files or software settings. If you are not familiar with this, it is safer to leave defaults.
